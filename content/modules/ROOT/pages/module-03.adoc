= CLI Access

== Logging In with the CLI

Open the terminal tab and log in to the OpenShift cluster:

[source,bash,subs=attributes]
----
oc login --insecure-skip-tls-verify -u {openshift_cluster_admin_username} {openshift_api_url}
----

When prompted, enter the password: `{openshift_cluster_admin_password}`

== Verifying the Cluster

Check the cluster nodes:

[source,bash]
----
oc get nodes
----

You should see a single node with roles `control-plane,master,worker`.

List all cluster operators:

[source,bash]
----
oc get clusteroperators
----

== Deploying a Sample Application

Create a new project:

[source,bash]
----
oc new-project test-app
----

Deploy a sample httpd application from source:

[source,bash]
----
oc new-app httpd~https://github.com/sclorg/httpd-ex.git
----

Watch the build progress:

[source,bash]
----
oc logs -f buildconfig/httpd-ex
----

Once the build completes, expose the service:

[source,bash]
----
oc expose service httpd-ex
----

Get the route URL:

[source,bash]
----
oc get route httpd-ex -o jsonpath='{.spec.host}'
----

You can open this URL in a new browser tab to see your deployed application.
