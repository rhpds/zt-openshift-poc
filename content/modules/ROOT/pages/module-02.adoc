= OpenShift Console

== Accessing the Web Console

The OpenShift web console is embedded in the *OpenShift Console* tab on the right side of your screen.

You can also open it directly in a new browser tab:

{openshift_console_url}

== Logging In

Use the following credentials:

[cols="1,2"]
|===
| Username | {openshift_cluster_admin_username}
| Password | {openshift_cluster_admin_password}
|===

== Exploring the Console

Once logged in, explore the following areas:

* *Home > Overview* -- cluster status, resource utilization, and health
* *Operators > Installed Operators* -- pre-installed operators on the cluster
* *Workloads > Pods* -- running pods across namespaces
* *Networking > Routes* -- exposed application endpoints

TIP: Switch between the *Administrator* and *Developer* perspectives using the dropdown in the top-left navigation.
